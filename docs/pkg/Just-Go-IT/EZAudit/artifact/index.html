func CensorGlobal ¶ 

func CensorGlobal(input string) (censored string)

CensorGlobal takes the string and replaces the text, based on the global censor regexp to "--censored--". Returns the sanitized string

func CensorLocal ¶ 

func CensorLocal(input string, reg []regexp.Regexp) string

CensorLocal takes string and specific regexp. Replaces text based on the specific regexp to "--censored--". Returns the sanitized string

func CheckLength ¶ 

func CheckLength(input string, step global.Step) string

CheckLength checks if the length is too long, if it is it, it saves an artifact with the input and returns the path to it

func CreateFolderIfNotExist ¶ 

func CreateFolderIfNotExist(path string)

CreateFolderIfNotExist creates an empty folder in the specified path, if the folder doesn't exist yet.

func SaveFile ¶ 

func SaveFile(path string, step global.Step)

SaveFile save File to the Artifact folder. The created fileName is based on getFilePath. The saved string depends on the censoring configuration and DontSaveArtifact parameter in the config-file.

func SaveString ¶ 

func SaveString(artifact string, step global.Step)

SaveString saves string as a .txt File in the Artifact folder. The created fileName is based on getFilePath. The saved string depends on the censoring configuration and DontSaveArtifact parameter in the config-file.

func SetUp ¶ 

func SetUp(regexExpression []string, maxOutputLength int)

SetUp compiles the passed regexExpressions and saves them for later use. If maxOuputLenght is defined in the config-file then it overrides the maxOutputLength (which is by default = 350). Set maxOutputLength to -1, if you want to set it to unlimited.
